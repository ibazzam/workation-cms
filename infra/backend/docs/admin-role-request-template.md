# Admin Role Request Template

Use this template when requesting admin access for a staff member.

## Copy/Paste Request

```markdown
### Admin Role Access Request

- Request date:
- Requested by (manager/team lead):
- Staff member name:
- Staff member email:
- Team:
- Temporary or permanent access: (Temporary / Permanent)
- If temporary, access end date:

#### Requested role (pick one)
- [ ] ADMIN_CARE (vendor care + property/workation troubleshooting)
- [ ] ADMIN_FINANCE (billing/accounting/payments operations)
- [ ] ADMIN_SUPER (full admin access)

#### Business reason (required)
- Why this access is needed:
- What tasks they must perform:

#### Minimum access check (required)
- Why a lower role will not work:

#### Risk & approvals
- Data/operations impacted:
- Approved by (name + role):
- Approval date:

#### Offboarding / review
- Next access review date:
- Removal plan when no longer needed:
```

## Quick selection help

- Choose **ADMIN_CARE** for vendor support and property/workation fixes.
- Choose **ADMIN_FINANCE** for reconciliation, payment job actions, and billing operations.
- Choose **ADMIN_SUPER** only when full cross-area access is truly required.

## Approval checklist

- [ ] Business reason is specific
- [ ] Lowest possible role selected
- [ ] Approver is identified
- [ ] Review date is set

## Quick chat/email version (copy/paste)

```text
Admin access request:
Staff: <name> (<email>)
Role: <ADMIN_CARE | ADMIN_FINANCE | ADMIN_SUPER>
Reason: <1 sentence business reason>
Duration: <Permanent or end date> | Approver: <name>
```
